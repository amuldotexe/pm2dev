# Local LLM Independence tools

A bunch of use cases where we need local LLMs because the existing LLM APIs will prohibit us from using them. A very good example of that is how decompilation systems work. We need a small LLM which is very good at decompilation, and that could be a native Tauri app. 

Tomorrow, the LLMs that are provided with APIs might even say, "Hey, you are not allowed to decompile this." In order to be free from that, I need to have some control over whether I want to decompile software or not, right?
